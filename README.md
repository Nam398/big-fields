# BIGFIELDS 2026 초대장 웹페이지

## GitHub Pages에 올리는 방법
1. GitHub에서 새 저장소를 만듭니다. 예: `bigfields-invitation`
2. 이 폴더 안의 `index.html`, `style.css`, `script.js`, `poster.jpg` 네 파일을 저장소 최상단에 업로드합니다.
3. 저장소의 **Settings → Pages**로 들어갑니다.
4. **Build and deployment**에서 `Deploy from a branch`를 선택합니다.
5. Branch는 `main`, Folder는 `/ (root)`로 설정하고 Save를 누릅니다.
6. 약 1~2분 뒤 `https://사용자이름.github.io/bigfields-invitation/` 형태의 주소가 생성됩니다.

카카오톡에서는 링크 미리보기만 정적으로 보일 수 있고, 링크를 눌러 페이지를 열면 실시간 카운트다운이 동작합니다.
